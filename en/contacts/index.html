How to write for a more useful reply

To help me understand faster, you can mention:

  • which device / board you use (Arduino, ESP, etc.);
  • what the project is for (fish finder, smart home, game, etc.);
  • what you have already done and where it breaks;
  • if possible — attach photos / screenshots (preferably via Telegram).

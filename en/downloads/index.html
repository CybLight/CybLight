Android

CybLight mobile app

Official Android app: CybLight account sign-in, friends list, private messages, and security settings.

Version 0.4.1 · Android 8.0+

Download

Download APK from site Download from GitHub Release page

Direct file on cyblight.org is not available yet — use the "Download from GitHub" button.

Features

  • Sign in via login.cyblight.org (username, password, Turnstile)
  • Two-factor authentication (2FA) support
  • Friends list with online status
  • Private messages and chats
  • Profile, security, sessions, and passkeys
  • Russian, Ukrainian, and English
  • Update check on launch

Installation

  1. Download the APK using the button above.
  2. Open the file on your phone.
  3. If Android warns about an unknown source — allow installation and tap "Install anyway".
  4. Launch CybLight and sign in to your account.

When updating over an older build, you may need to uninstall the previous version if the signing key differs.
